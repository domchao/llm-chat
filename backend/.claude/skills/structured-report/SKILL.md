# Structured Report Format

Create professional reports following this standardized structure.

## Report Structure

### 1. Executive Summary

-   Key findings (2-4 bullets)
-   Main recommendations
-   Critical actions needed
-   Keep to 3-4 sentences maximum

### 2. Methodology (if applicable)

-   Data sources
-   Analysis approach
-   Timeframe covered
-   Limitations or assumptions

### 3. Detailed Findings

Organize by theme or priority:

-   Use clear section headings
-   Lead with the insight, not the data
-   Support claims with specific numbers
-   Include visualizations when helpful

### 4. Recommendations

Prioritized action items:

-   **High Priority**: Immediate actions needed
-   **Medium Priority**: Important but not urgent
-   **Low Priority**: Nice-to-have improvements

### 5. Conclusion

-   Reiterate key points
-   Next steps
-   Timeline if applicable

## Writing Guidelines

**Be Direct**: Lead with conclusions, not buildup
**Be Specific**: "Sales dropped 23% in Q3" not "Sales declined significantly"
**Be Actionable**: Every finding should lead to a recommendation
**Be Structured**: Use consistent formatting, numbered lists, clear headers
